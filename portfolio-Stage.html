Informations sur le projet

Énoncé : Transformer un fichier Excel automatisé qui permet de faire un compte rendu quotidien des évènements survenus dans la journée sur le réseau en application web pour permettre une meilleure portabilité et le fiabiliser. Mais aussi le rendre plus simple d'utilisation, ainsi il sera plus facile de le partager ou de l'imprimer.


Mes Objectifs :

  • Création d'une application avec le framework Symfony en php, capable de renseigner toute les informations nécéssaire au compte rendu quotidien des évènements survenus dans la journée sur le réseau.
  • Capable d'importer ces informations depuis un fichier csv.
  • Capable d'exporter les informations sur l'application en document Excel pour pouvoir l'imprimer.
  • Gérer les archives
  • La connexion s'effectue avec Guardien.
  • Mise en place de serveurs de production et de developpemnt.
  • Nombreux audit de code pour vérifier et corriger les risques de sécurités grâce à CheckMarx.
  • Faire des tests unitaires avec PHP Unit pour vérifier que toute mes fonctions marche correctement
  • Faire une doc utilisateur et une doc technique
  • Faire les logs
